## A quoi sert le certificat de non-gage ?

Le certificat de situation administrative (souvent appele non-gage) permet de verifier s’il existe un obstacle administratif a la cession.  
Dans un dossier d’epave, il rassure toutes les parties et evite les blocages de derniere minute.

## Quand le demander avant un enlèvement d’epave ?

Demandez-le **avant** de fixer le creneau si :

- le vehicule a change de main plusieurs fois ;
- la carte grise est ancienne ou mal renseignee ;
- vous voulez securiser une cession avec rachat.

## Ou generer le non-gage en ligne

Lien officiel : [Generer un certificat de non-gage](https://siv.interieur.gouv.fr/map-usg-ui/do/csa_retour_dem_certificat).

## Informations a preparer

Preparez :

- numero d’immatriculation ;
- date de premiere immatriculation ;
- identite du titulaire selon la carte grise.

## Erreurs frequentes

Les erreurs les plus courantes viennent d’une saisie inexacte ou d’une carte grise ancienne avec format different.  
Si vous avez un doute, transmettez d’abord les photos de vos documents pour validation.

## A retenir

Le non-gage n’est pas qu’une formalite : il fluidifie la cession et evite les aller-retours.
